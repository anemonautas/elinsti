# Anemonautas Learning Center.

# Donde estoy?

Ahora estas en la pagina de GitHub del projeto `insti`. Esto es un proyecto opensource, sponsorizado por Anemonautas Learning Center, 
un centro educativo virtual que desea acercar esta herramienta al alumnado para proporcionar y facilitar el uso de una herramienta 
que les sera util en el futuro a medio plazo.
